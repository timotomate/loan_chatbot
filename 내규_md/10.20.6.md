10.20.6 대출금리 및 보증료율<신설 2018.2.8, 2018.6.18>

1. 채무자 부담 이자율 : 대출금리 + 보증료율(보증승인과 일치)

(1) 대출금리(은행) : 4.0%(고정금리)

(2) 보증료율(기금) : 국민행복기금이 자체 CSS등급에 따라 차주별 차등 적용(8%~20%)

| CSS등급 | 특수<br>채무자 | 1등급 | 2등급 | 3등급 | 4등급 | 5등급 | 6등급 | 7등급 | 8등급 | 9등급 | 10등급 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 대출금리 | 4% | 대출거절 |  |  |  |  |  |  |  |  |  |
| 보증료율 | 8% | 11% | 14% | 17% | 20% |  |  |  |  |  |  |
| 등급 | S0 | S1 | S2 | S3 | S4 | S5 | N |  |  |  |  |

* 특수채무자: 기초수급자, 차상위계층, 한부모가족 등

2. 성실상환자 금리감면

(1) 적용방법: 대출 실행 후 매 6회차(6개월) 경과시점 마다 원리금 연체없이 정상 납부한 경우 감면 적용

※ 예시) 6회차 연체로 미감면시 7회차 원리금 납부일 전날까지 연체가 해소되면 7회차 부터 감면 적용, 7회차 납부일에 연체 해소시 8회차 부터 감면 적용

(2) 감면금리 차감 방법 : 보증료율에서 등급별 차등 차감

| 등 급 | S0,S1 | S2 | S3 | S4 | S5 |
| B. 최초 2년(최대4회) | 1.0 % | 1.5% | 2.0% | 2.5% | 3.0% |
| C. 최초 2년 경과 후<br>2년 6개월(최대5회) | 0.3% |  |  |  |  |
