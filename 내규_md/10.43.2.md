10.43.2 대출대상

서금원에서 보증서 발급이 가능한 만 34세 이하의 아래 대상자

(서금원에서 자격요건 심사 및 보증약정 완료되어 은행에서 별도의 검증절차 없이 진행가능)

| 구분 | 자격요건 |  |
| --- | --- | --- |
| 취업준비생 | 대학(원)생 | 대학(원)에 재학·휴학·졸업유예 중인 자 |
|  | 학점은행학습자 | 학점인정기관 수강중이며, 12학점 이상 수강한 이력이 있는 자 |
|  | 미취업청년 | 직장에 재직 중이지 않은 자(직장건강보험 가입여부로 판단) |
| 사회초년생 | 중소기업에 1년 이하 재직 중인 자<br>※단시간 기간제근로자는 취업중비생 중 한가지 자격으로 인정 |  |
| 청년사업자 | 사업장등록증의 개업일로부터 1년이하 사업자로 연소득 35백만원 이하인 자(미등록 사업자는 미취업청년 자격으로 인정) |  |
